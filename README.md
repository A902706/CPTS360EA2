# CPTS360EA
This is the repository for our group project for the hardware hackathon

Team members: 
Andres S, Jholyn C, Karen T, Andrea C

So we as a team decided we wanted to create a automatic waterling system for a plant. The idea was if the soil moisture became too low, then the pump would activate and push water to the plant to keep the plant at the right moisture level. This will help our little friend live a long life full of water and not dry out like a couple of my plants currently in my dorm room. 

We used a Moisture sensor, water pump, matrix lights, and a Arduino.
Along with lots of documentation and the Arduino IDE.
